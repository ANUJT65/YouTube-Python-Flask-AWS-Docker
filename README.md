# AWS PYTHON FLASK DOCKER

#### SSH Commands
* change permission of the `.pem` file
    ```bash
    chmod 0400 .pem file path
    ``` 
* connect to remote server using ssh
    ```bash
    ssh ec2-user@public_ip
    ```
    in case you choose `Ubuntu` os then
    ```bash
    ssh ubuntu@pubic_ip
    ```

#### Install Packages
* update the system
